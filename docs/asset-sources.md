## Good Next Downloads

- OpenGameArt visual novel character sprite: https://opengameart.org/content/visual-novel-character-sprite
  - License shown as CC0.
  - Useful if we want a no-attribution-required alternate placeholder.
- OpenGameArt visualnovel/anime/manga CC0 collection: https://opengameart.org/content/visualnovelanimemanga-cc0
  - Curated CC0 links for portraits, rooms, and visual-novel-adjacent assets.
- itch.io free VN backgrounds: https://itch.io/game-assets/free/genre-visual-novel/tag-backgrounds
  - Download manually from each creator page and keep their individual terms.
- itch.io free VN sprites: https://itch.io/game-assets/free/genre-visual-novel/tag-sprites
  - Good for quickly finding expression sets.
- Sutemo female character sprite: https://sutemo.itch.io/female-character
  - Page says personal/commercial use is allowed, but the sprite cannot be resold on its own.
- NoranekoGames Aiko sprite: https://noranekogames.itch.io/aikosprite
  - Free visual novel sprite pack. Download through itch.io.
- LoudEyes dating sim UI pack: https://loudeyes.itch.io/dating-sim-ui-pack
  - Useful for menu, textbox, and save/load UI direction.
- itch.io textbox assets: https://itch.io/game-assets/free/tag-textbox
  - Good for testing alternate dialogue box skins.

## Private Reference Packs

Do not commit commercial game asset dumps. If a private local reference pack is needed, put it outside the repo and import it through the app's theme-folder picker.
