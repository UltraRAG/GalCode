# Senren Banka Visual Notes

This file is a reference-only style analysis for GalCode. It must not contain copied images, extracted CGs, or redistributed original assets.

## Useful Direction

- Overall mood: warm Japanese summer romance, shrine-town calm, light comedy, polished visual novel presentation.
- Interface feel: clean character-forward layout, translucent dialogue surfaces, clear nameplates, compact navigation.
- Color movement: warm wood and lantern tones balanced by cool sky or teal accents; avoid a single-hue palette.
- Composition: character art dominates the emotional center while UI stays readable in the lower third.
- Character system: each heroine needs a stable silhouette, several expressions, and an outfit language that implies role/personality.

## GalCode Translation

- Keep the default theme original and wa-koi inspired rather than derivative.
- Use developer-world details as subtle props: laptop, terminal glow, code notebooks, diff papers, task charms.
- Separate `Agent`, `Character`, and `Theme` so a user can bind any agent to any heroine later.
- Prefer generated or commissioned original assets for distributed builds.

## Do Not Do

- Do not download official CGs, character sprites, screenshots, UI parts, icons, music, or logos into this repo.
- Do not make the default characters recognizable as Senren Banka characters.
- Do not ship third-party image-board assets.
